# slidable_text_field
A text field combined with a slider to make user experince smooth inspired by [Nussi Einhorn](https://www.linkedin.com/in/nussi-einhorn/) design
### Supported Currencies:
[x] usd
[x] irr
[x] euro
[x] plain number
